webapp2를 사용하여 진행합니다.. 기존의 코드는 webapp입니다.. 

````python

import webapp2


class MainPage(webapp2.RequestHandler):

    def get(self):
        self.response.headers['Content-Type'] = 'text/plain'
        self.response.write('Hello, webapp2 World!')


application = webapp2.WSGIApplication([
    ('/', MainPage),
], debug=True)

````
